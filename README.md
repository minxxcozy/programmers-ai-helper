# 🤖 programmers-ai-helper
> AI 기반 코딩 어시스턴트 사용자 스크립트

## ✔️ 개요  
`programmers-ai-helper`는 웹 브라우저 상에서 코딩 관련 AI 도우미 기능을 추가해주는 사용자 스크립트 모음입니다. 
개발자는 프로그램밍 문제 풀이 사이트 혹은 코딩 학습 환경에서 본 스크립트를 설치하면 보다 빠르고 편리하게 AI 어시스턴트 기능을 활용할 수 있습니다.

## ⚙️ 주요 기능  
- AI 코드 생성 및 추천  
- 코드 번역 / 리팩토링 보조  
- 웹페이지 내에서 직접 AI 대화 인터페이스 삽입  
- 기타 코딩 작업을 보조하기 위한 유틸리티 스크립트들

## 🛠️ 설치 방법  
1. 브라우저에 Tampermonkey 확장(Add-on)을 설치합니다.  
2. 본 저장소의 `build/` 폴더로 이동합니다.   
3. Tampermonkey에서 **새 스크립트 추가** → 붙여넣기 혹은 직접 URL로 설치합니다.  
4. 설치 후 해당 코딩 사이트를 열면 스크립트가 자동으로 동작합니다.

> ✅ **build 폴더의 스크립트를 Tampermonkey에 넣어서 사용하면 됩니다.**

## 📢 사용 방법  
- 스크립트 설치 후 웹페이지 상단 또는 지정된 위치에 AI 대화창이나 버튼이 나타납니다.
- 버튼 클릭 시 입력창이 나타나고, 여기서 자연어로 코드 생성 요청 또는 설명 요청이 가능합니다.
- 생성된 코드나 설명을 복사해서 직접 프로젝트나 풀이 환경에 붙여넣습니다.
- 필요 시 옵션 (언어 선택, 코드 스타일 지정 등)을 설정할 수 있습니다.

## 💡 참고  
- 본 스크립트를 사용함으로써 발생하는 문제(브라우저 충돌, 사이트 기능 장애 등)에 대해 책임을 지지 않습니다.  
- 사이트 이용 약관을 위반할 수 있으므로, 사용 전에 해당 코딩 사이트의 정책을 확인하세요.
